# NotaBot Frontend

Frontend oficial de NotaBot, una plataforma web enfocada en la gestión e interacción inteligente de información académica mediante una interfaz moderna, rápida y escalable.

**Requicitos
- node js
- git

**Instalación y ejecución

1. Clonar el repositorio
git clone https://github.com/santicar10/notabot-frontend.git

3. Entrar al proyecto
cd notabot-frontend

5. Instalar dependencias
npm install

7. Ejecutar en modo desarrollo
npm run dev
